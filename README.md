# Domestic Violence Silent Alert System

A safety-focused application designed to help victims in dangerous situations.

## Features
- Silent emergency alert using a secret key.
- Sends help requests to trusted contacts.
- Desktop application developed using Java.
- Currently being enhanced for mobile devices.

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Desktop Application Development

## Future Improvements
- Mobile application version
- GPS location sharing
- Multiple trusted contacts
- Improved user interface
